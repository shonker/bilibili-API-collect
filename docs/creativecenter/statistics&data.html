                # 视频数据增量趋势

                https://member.bilibili.com/x/web/data/article/thirty

                请求方式：GET

                认证方式：仅可Cookie（SESSDATA）

                数据为前30天

                url参数：

                参数名 类型 内容 必要性 备注
                type num 目标数据类型 必要 类型代码见下表

                类型代码type：

                代码 含义
                1 播放
                2 弹幕
                3 评论
                4 分享
                5 投币
                6 收藏
                7 充电
                8 点赞

                json回复：

                根对象：

                字段 类型 内容 备注
                code num 返回值 0：成功
                -101：账号未登录
                -400：请求错误
                message str 错误信息 默认为0
                ttl num 1 作用尚不明确
                data array 趋势列表

                data数组：

                项 类型 内容 备注
                0 obj 1天前的数据
                n obj （n+1）天前的数据
                …… obj …… ……
                29 obj 30天前的数据 最后一条

                data数组中的对象：

                字段 类型 内容 备注
                date_key num 对应时间 时间戳 前一天的8:00
                total_inc num 增加数量 意义为数据类型决定

                示例：

                查询30天前的视频播放增量趋势，可知2020-04-05的播放增量为46，2020-04-04的播放增量为58

                curl -G 'https://member.bilibili.com/x/web/data/pandect' \
--data-urlencode 'type=1' \

                # 稿件操作来源占比情况

                https://member.bilibili.com/x/web/data/survey

                请求方式：GET

                认证方式：仅可Cookie（SESSDATA）

                数据为上一天的

                字段 类型 内容 备注
